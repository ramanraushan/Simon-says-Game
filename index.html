Simon Says Game

How to Play

  • 🎯 Press "Start Game" to begin.
  • 👀 Watch the flashing color sequence.
  • 🧠 Repeat the sequence by clicking the same color buttons.
  • ✅ Each level adds one new color to the sequence.
  • ❌ If you press the wrong button, the game ends.
  • 🔁 Reload page to restart after Game Over.
